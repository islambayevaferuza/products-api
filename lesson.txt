SQLite nima?

SQLite - bu yengil (lightweight) ma'lumotlar bazasi.

oddiy qilib aytganda:

    Array => RAM'da saqlanadi
    SQLite => Faylda saqlanadi

Masalan: 

    const products = [];

Bu ma'lumot server restart bo'lsa yo'qoladi.

SQLite esa:

    database.sqlite

fayliga ma'lumotlarni yozadi va unga saqlanib

SQLite qanday ishlaydi?

Biz:

    db.run(...)
oraqali ma'lumot yozamiz.

    db.get(...)
oraqali bitta ma'lumot olamiz.

    db.all(...)
oraqali ko'p ma'lumot olamiz.
